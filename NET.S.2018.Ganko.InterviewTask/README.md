Задание для технического интервью
Практическая часть.
1. Реализуйте указанный алгоритм. Оформите программу в виде
консольного приложения. Дайте оценку времени выполнения
алгоритма в зависимости от длины исходных данных.
* сортировка слиянием;

2. Реализуйте заданную структуру данных в виде класса (набора
классов). Для внутренней структуры класса-коллекции использовать
только одномерные массивы(!).
* очередь строк;

3. Для структуры данных из п. 2 напишите обобщенную реализацию
(Generics), позволяющую хранить объекты (ссылочные переменные).
